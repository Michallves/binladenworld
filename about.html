Jogo Base em JavaScript

"Bin Laden World" é um jogo de plataforma onde você controla o personagem BinLaden.
O objetivo é evitar colidir com as cadeiras que se movem em direção ao personagem.
O jogo apresenta uma tela inicial com o título e um botão "Start" para iniciar.

  • Durante o jogo, a velocidade dos Cadeiras aumenta progressivamente, acumulando pontos conforme o BinLaden permanece vivo.
  • Colisões com as cadeiras resultam em um fim de jogo, exibindo uma tela de "Game Over" com a opção de reiniciar.
  • O jogo também possui animações de nuvens em movimento e é desenvolvido usando HTML, CSS e JavaScript

O jogo foi desenvolvido para a Avaliação de Programeção WEB I, ministrada pela professora Elisângela Reboucas;
